## Core claim

Project setup should be encoded and discoverable so common work does not depend
on remembered local machine steps.

## Allowed material

- Pinned runtimes
- Discoverable tasks
- Consistent commands for humans, CI, and agents
- Custom tooling build steps behind stable project commands

## Avoid

- Private repository names
- Machine-specific paths
- Exact internal scripts
- Work-specific setup history
